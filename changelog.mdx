## Release History

<Update label="2024-10-15" description="v2.1.0" tags={["feature","improvement"]}>

## New Features

- Added real-time collaboration editing for up to 50 users simultaneously.
- Introduced advanced search with full-text indexing across all projects.

## Improvements

- Enhanced performance for large documentation spaces with lazy loading.
- Updated UI components for better mobile responsiveness.

## Bug Fixes

- Resolved sync issues during high-traffic sessions.
- Fixed export functionality for embedded images.

</Update>

<Update label="2024-09-20" description="v2.0.0" tags={["breaking","feature"]}>

## New Features

- Launched API endpoints for programmatic document management.
- Integrated webhook support for external notifications.

## Breaking Changes

- Updated authentication flow to use JWT tokens instead of sessions.
- Removed legacy markdown parser in favor of a more robust engine.

## Improvements

- Streamlined invitation process with bulk user imports.
- Added analytics dashboard for page usage tracking.

</Update>

<Update label="2024-08-10" description="v1.0.0" tags={["feature"]}>

- Core documentation editor with markdown support.
- Basic project organization using folders and pages.
- User authentication and simple sharing features.

## Improvements

- Initial setup wizard for new users.
