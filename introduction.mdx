## Welcome to aishwarya

You use aishwarya to create and manage comprehensive documentation for your projects. This platform provides tools for organizing content, collaborating with teams, and integrating with APIs for seamless workflows. Start by exploring the key features below to build your documentation space efficiently.

<Columns cols={3}>
  <Card title="Organize Content" icon="file-text" href="#">
    Structure your documentation with folders, pages, and hierarchical navigation. Use tags and search to keep everything accessible.
  </Card>
  <Card title="Collaborate Easily" icon="users" href="#">
    Invite team members, assign roles, and track changes in real time. Share links or export content for external use.
  </Card>
  <Card title="Integrate APIs" icon="code" href="#">
    Connect with external services via webhooks and custom integrations. Automate updates and sync data across platforms.
  </Card>

## Quick Start Guide

Follow these steps to set up your first documentation project.

<Steps>
  <Step title="Create Account" icon="user">
    Sign up with your email and verify your account. This gives you access to unlimited projects.
  </Step>
  <Step title="New Project" icon="plus">
    Click the `+` button to start a new documentation space. Name it and add an initial description.
  </Step>
  <Step title="Add Pages" icon="file-text">
    Create your first page using the editor. Include markdown, images, and code snippets for rich content.
  </Step>
</Steps>

<Callout kind="info">
  Remember to define your project's scope early. This helps in maintaining consistent documentation.
</Callout>

## Platform Features

Explore different ways to use aishwarya across environments.

<Tabs>
  <Tab title="Web" icon="globe">
    Access the full editor via browser. Supports real-time previews and version history.
    
    ```javascript
    // Example: Embedding a page
    const embedUrl = 'https://aishwarya.com/embed/page-id';
    document.getElementById('embed').src = embedUrl;
    ```
  </Tab>
  <Tab title="Mobile" icon="phone">
    Use the app for on-the-go edits. Sync changes instantly with the web version.
    
    ```javascript
    // API call for mobile
    fetch('/api/mobile/sync', { method: 'POST', body: JSON.stringify({pageId: '123'}) });
    ```
  </Tab>
</Tabs>
